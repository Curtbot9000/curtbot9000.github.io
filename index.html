Salem Mod Loader

4th August 2023

I developed a comprehensive modloader and modmanager for the game Town of Salem 2, leveraging various technologies to ensure a seamless experience for both developers and users. Key features include:

  • Automatic Updates: Implemented through a simple API hosted on GitHub, this feature ensures that both the modloader and the mods it manages are always up-to-date.
  • Built-in Mod Browser: Allows users to easily browse and install mods directly from within the game.
  • Dynamic Settings System: Enables mod developers to add customizable settings to their mods. These settings can trigger code execution as values are updated, such as instantly swapping custom sky themes in the game.
  • Automatic Mod Disabling: Identifies and disables outdated mods that may break due to game updates, ensuring stability.
  • Malicious Mod Detection: The modloader includes a security feature that allows me to externally mark malicious mods, uninstalling them for users before they can be loaded into the game. This ensures a safe environment for all users.

All UI elements for the modloader were designed in Unity and integrated into the game via asset bundles.

  • HarmonyLib
  • Bepinex
  • GitHub

BetterTOS2

26th January 2024

BetterTOS2 is a mod for the game Town of Salem 2 that I developed alongside TubaAntics. It introduces numerous new roles, a full new gamemode selecting system, rebalances the game and more. The mod has received over 2.5 million views on youtube with that number growing every week. The quality of the mod was so high that the developers for Town of Salem 2 brought me onto the team to work on the actual game.

In order to create the mod, we had to program our own server for users to connect to including having to recode all of the original game's netcode. Our server was able to support over 300 concurrent users on a VPS with only 1 CPU core, and 4gb of Memory.

I developed a Discord Bot for the mod that would allow users to connect their BetterTOS2 accounts with their Discord acounts. This allowed users to track their ingame stats through the bot, and allowed users to talk in the mod's in-game chat through a discord channel. Ontop of this it provided us with a very handy interface for remotely moderating.
